# micro-sb3-Microservices-with-Spring-Boot-3-

Overview
Microservices-based project built in VS Code using Spring Boot 3.x and Spring Cloud.

Key Features - Service Registry (Eureka Server): Service discovery.
API Gateway: Routes client requests.
Inter-Service Communication: REST via API Gateway.
Scalable Design: Independent services.

Technologies Used - Java 17, Spring Boot 3.x, Spring Cloud, Eureka Server, Spring Cloud Gateway, Maven, H2/MySQL.

Testing - Verified in Eureka Dashboard.
          Tested APIs via Postman
